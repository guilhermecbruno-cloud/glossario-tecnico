Tags HTML Comuns

Essas são algumas das tags mais usadas para criar páginas web.

  • <div> divisão genérica de conteúdo
    <div>Texto</div>
  • <p> parágrafo de texto
    <p>Olá mundo</p>
  • <h1> a <h6> títulos
    <h1>Título Principal</h1>
  • <img> imagem
    <img src="foto.jpg" alt="foto">
  • <a> link
    <a href="https://exemplo.com">Clique aqui</a>
  • <ul> / <ol> listas (não ordenadas / ordenadas)
    <ul><li>Item</li></ul>
  • <table> tabela
    <table><tr><td>Dado</td></tr></table>
  • <input> campo de entrada
    <input type="text">
  • <form> formulário
    <form>...</form>
  • <section> seção de conteúdo
    <section></section>
