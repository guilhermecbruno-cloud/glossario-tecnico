Glossário de JavaScript

Alguns comandos básicos de JavaScript usados no dia a dia do programador.

  • alert() mostra uma janela de alerta
    alert("Bem-vindo!");
  • console.log() escreve no console
    console.log("teste");
  • document.querySelector() seleciona um elemento
    document.querySelector("p");
  • addEventListener() adiciona evento
    botao.addEventListener("click", funcao);
  • function declara uma função
    function saudacao() { alert("Oi"); }
  • setInterval() repete com intervalo
    setInterval(minhaFunc, 2000);
  • setTimeout() espera para executar
    setTimeout(() => { }, 1000);
  • let / const variáveis
    const nome = "Ana";
  • if / else condição
    if (x > 5) { ... }
  • typeof verifica tipo
    typeof 123
