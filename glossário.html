Glossário de CSS

Veja abaixo algumas propriedades CSS úteis para estilizar uma página web.

  • color muda a cor do texto
    color: blue;
  • background-color muda a cor do fundo
    background-color: lightgray;
  • margin espaço externo
    margin: 10px;
  • padding espaço interno
    padding: 20px;
  • font-size tamanho da fonte
    font-size: 16px;
  • border cria bordas
    border: 1px solid black;
  • text-align alinha o texto
    text-align: center;
  • display modo de exibição
    display: flex;
  • justify-content alinha itens
    justify-content: space-between;
  • position posicionamento
    position: absolute;
